# Running Rework
## Prerequisities
- Docker
- Bombd (by ennuo)
- Node.js
- At least 2GB of RAM
- A PostgreSQL Database
- A functioning brain (obviously)

## How to run it

First, rename .env.example to .env and set it up.
Then, open terminal on the project root, type `yarn install` to install the dependencies and type
`docker compose up --build` to run Rework.

You also need [Bombd](https://github.com/ennuo/bombd) running in the background for matchmaking.
